# 무료체험(7일) 설정 가이드

스토어 연동 무료체험(7일) → 체험 종료 후 자동구독 전환 기능의 운영 가이드.

체험을 실제로 **부여하는 주체는 스토어(App Store / Google Play)**입니다.
앱/서버는 "체험 중" 상태 표시와 전환 유도(종료전 푸시)만 담당합니다.
따라서 스토어 콘솔 설정 없이는 코드만으로 체험이 시작되지 않습니다.

## 동작 흐름

1. 사용자가 `7일 무료체험 시작 · 이후 3,900원/월` 버튼 탭
2. 스토어 결제 시트 → **0원 결제**(Face ID/비밀번호 확인만, 실제 과금 없음)
3. 7일간 프리미엄 이용 (`inTrial = true`)
4. 종료 약 48시간 전 전환 유도 푸시 1회 발송
5. 7일 뒤 스토어가 **자동으로 정기결제 전환**(auto-renew) → 프리미엄 유지

> 체험 시작 시 사용자가 스토어 결제 시트에서 1회 "확인"을 눌러야 합니다
> (Apple/Google 정책 — 결제수단 등록·동의 없이 자동과금 불가). 돈은 안 나갑니다.

---

## 코드에 이미 반영된 것 (추가 작업 불필요)

- **서버**
  - `AppConfig.trialEnabled` / `trialDays`(기본 7) — 원격 설정, `/api/admin/app-config/public`로 노출
  - `Subscription.trial_ending_notified_at` — 종료전 푸시 중복 방지 컬럼
  - `SubscriptionsService.sweepTrialEndingNotifications()` — 매시간 cron, 체험 종료 ~48h 전 전환 유도 푸시 1회
  - `NotificationsService.notifyUser()` — 특정 유저 푸시 헬퍼
  - DB 컬럼은 부팅 시 onModuleInit ALTER로 자동 추가(synchronize off 대응)
- **앱**
  - paywall 버튼/안내 문구가 `trialEnabled` ON일 때 "N일 무료체험" 으로 전환
  - 체험 중 배너·관리 카드에 "체험 N일 남음" 카운트다운
  - `AppRemoteConfig.trialEnabled` / `trialDays` 수신

## ✅ 직접 해야 하는 부분 (체크리스트)

### 1. 스토어 콘솔에 무료체험 offer 등록 (심사 시간 소요)

- [ ] **App Store Connect**
  - `lingoloop_premium_monthly` 구독 → **Introductory Offer** 추가
  - 유형 `Free`, 기간 `1 Week`, 대상 `신규 구독자(New subscribers)`
- [ ] **Google Play Console**
  - 해당 구독의 base plan → **Offer** 추가
  - 무료체험(Free trial) phase `7일`, eligibility `신규 고객(이 구독 미경험자)`

> ⚠️ offer 기간을 7일 이외로 바꾸면, 4번 백스테이지의 `무료체험 일수`도 동일하게 맞춰야 표시가 정확합니다.

### 2. 서버 배포

- [ ] 서버 배포 (부팅 시 DB 컬럼 추가 + cron 자동 적용)

### 3. 앱 신규 빌드 배포 (심사 시간 소요)

- [ ] 변경된 paywall 문구·카운트다운 포함 새 빌드 업로드 → 스토어 심사 통과

### 4. 백스테이지에서 토글 ON (1·2·3 모두 라이브된 뒤)

- [ ] 백스테이지 → **결제 상품 설정**
  - `빌링 사용` ON
  - `무료체험 노출` ON
  - `무료체험 일수` = 7 (스토어 offer와 동일)

> 토글은 스토어 offer + 새 앱이 모두 라이브된 후 켤 것. 그래야 사용자가
> "무료체험" 문구를 보고 눌렀을 때 실제로 체험이 부여됩니다.

## 🧪 테스트해야 하는 부분

### 스토어 테스트 계정 준비

- [ ] iOS: App Store Connect **Sandbox 테스터** 계정
- [ ] Android: Play Console **라이선스 테스터** 계정

### 체험 시작 플로우

- [ ] `7일 무료체험 시작` 버튼 노출 확인 (`trialEnabled` ON 상태)
- [ ] 버튼 탭 → 스토어 시트에서 **0원** 표시 확인 → 구매 완료
- [ ] 구독 화면 배너가 `무료 체험 중 · 체험 N일 남음 · (날짜)부터 자동 결제` 로 표시
- [ ] 프리미엄 기능 잠금 해제 확인 (문장 퀴즈, 복습 큐 무제한, 광고 제거 등)

### 자동 전환 / 종료전 푸시

- [ ] 종료전 푸시(`무료 체험이 곧 끝나요`) 수신 확인
  - 샌드박스는 체험/갱신 주기가 단축됨(예: iOS 7일 → 수 분). cron은 매시간이므로
    실제 푸시 검증은 production-like 환경이나 짧은 offer로 확인 권장.
- [ ] 체험 종료 후 **자동으로 정기결제 전환** → 프리미엄 유지 확인
- [ ] 갱신 후 배너가 `다음 결제일 · (날짜)` 로 바뀌는지 확인

### 해지 케이스

- [ ] 체험 중 스토어에서 자동갱신 해지 → 만료일까지 프리미엄 유지, 이후 무료 강등 확인
- [ ] 해지 시 배너가 `구독 만료 예정 / (날짜)부터 무료 플랜으로 전환` 로 표시

## 롤백 / 끄는 법

- 백스테이지에서 `무료체험 노출` OFF → 앱 paywall이 기존 `프리미엄 시작하기` 문구로 복귀
  (이미 체험 중인 사용자의 스토어 구독에는 영향 없음 — 표시 문구만 변경)
